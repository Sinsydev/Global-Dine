 🍽️ Global Dine

A modern global restaurant menu platform with real-time currency conversion for international customers.

🌍 Overview

Global Dine is a modern web application that helps restaurants serve a global audience by converting menu prices into local currencies in real time.

It solves a real-world problem:
👉 International customers struggling to understand menu prices while traveling or ordering online.

With Global Dine, users can instantly view prices in their preferred currency using live exchange rates.

✨ Features.

🌍 Real-Time Currency Conversion.
🍽️ Restaurant-Focused UI.
⚡ Fast and responsive React interface.
💱 Supports multiple global currencies.
📊 Live exchange rate API integration.
🧾 Conversion history tracking.
🎨 Modern glassmorphism UI design.
📱 Fully responsive (mobile + desktop).
🔁 Multi-page routing (Home, About, Contact).


🧠 Problem It Solves

Restaurants lose international customers because:

Prices are confusing in foreign currencies
Lack of transparency in conversions
Poor user experience for travelers

Global Dine fixes this by:

Converting menu prices instantly
Improving customer trust
Making restaurants globally accessible

⚙️ Installation

Clone the repository:

git clone https://github.com/Sinsydev/global-dine.git

Install dependencies:

npm install

Run development server:

npm run dev

Build for production:

npm run build
🌐 Environment Setup (Optional)

If using API keys in future:

VITE_API_URL=https://api.exchangerate.host
